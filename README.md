# TQQNTA-ThemeCustm

Tenbuck-QQNT-Android Theme Customization/疼逊QQNT-Android主题自定化

&nbsp;&nbsp;&nbsp;&nbsp;_研究基于经验/qqnt-android-v9.1.25(or early)/qqnt2-android-v9.2.60_

我从最开始研究qq主题破解大抵是在23~24年 在持有root权限后多多少少什么都得想试试吧。。<br>
当时应该是有一篇帖子启发了我这么做 其中有这样一段<br>
"qq在你购买之前就已经预载了主题在本地，随时准备，等待你付款后通过某个trigger启用它"(目前已知在QQNT2中该问题被修复,见下文)<br>
是的 所以当时一个很搞的是 ~~我在基于那篇帖子研究的同时爬了一堆主题下来~~

我本来其实是考虑过 例如将方法和把爬下来的主题开一个repo然后上传到库, 是的 是真的<br>
很遗憾的是 因为一直以来的精神状态和日程上的应不暇接 这个项目被放弃直到了6202年(但我真的是想过做的

基于v9.2.60 发现qq现在可能不再通过或进行预载 而是在购买后"入库"到你的账户内 通过用户的手动点击来代替自动化trigger你购买的主题启用<br>
纯粹暴力且~~便捷~~(不一定)啊。。。

很难讲这种方式 但这也意味着将无法再尝试通过假购买把预载通过读data的方式导出来 想要一个新主题除了购买毫无办法

该repo仅提供一部分研究与示例展示 示例来源于QQNT主题商店手动爬取<br>
